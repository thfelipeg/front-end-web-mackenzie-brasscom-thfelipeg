Introdução ao HTML

> HyperText Markup Language

O HTML, diferente de outras linguagens, não é uma linguagem de programação, e sim uma linguagem de marcação, permitindo ao desenvolvedor construir páginas inteiras, estruturando seções, criando parágrafos, cabeçalhos, etc.

Tags, elementos e atributos

As tags são estruturas de linguagem de marcação, tendo uma marca de início e uma de fim, as quais contém instruções para que o navegador possa renderizar a página. A partir das tags surgem os elementos, eles são formados a partir de uma tag de abertura e uma de fechamento, por exemplo: <p>Parágrafo<p> , é um elemento.
