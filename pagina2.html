Imagens

O <img>, atributo de imagem, permite que o desenvolvedor insira imagens na página a partir de um arquivo local, ou links externos para a visualização. Escrito em código ficaria: <imghref="www.linkdaimagem">

HiperLink

Em português, hiperligação, é um atributo que permite a "conexão" entre qualquer elemento de uma páginas web que faça referência a outro link, podendo referênciar emails, telefones, e outras partes dentro do próprio site. Escrito como <a href="conteúdo">

Exemplo:

  • E-mail
  • Mackenzie

Listas

Tão simples quanto parece, as listas acrescentam a página de forma visual, organizando e listando conteúdos desejados, podendo ser feita a partir dos seguintes atributos: <ul> , <ol> e <li>

(O <li> é usado para representar um item que faz parte de uma lista, sendo contido dentro de um elemento pai, <ul> ou <ol>)

Exemplos:
